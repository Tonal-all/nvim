## Setup
0. Use the latest version of NVIM
1. Install `packer.nvim`
2. To clone this repository to your local machine is usually `~/.config/nvim`:
3. In neovim command line,run `:PackerInstall` to install the plugins

```
git clone https://github.com/Tonal-all/nvim.git ~/.config/nvim
```
